# 실시간 해상객체 검출 및 선박화재 감지

인공지능 및 빅데이터 팀프로젝트 <br>
2022.03.20 ~ 2022.06.19

<br>

1. 해상 객체 이미지와 화재 이미지를 수집해 Roboflow를 이용해 Annotation
2. Google colab에서 yolov5를 이용해 데이터 학습
3. yolov5로 학습한 모델을 tenserflow.js 파일로 변환
4. html로 웹 페이지 구현

<br>
